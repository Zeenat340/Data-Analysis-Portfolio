- **Tool:** Microsoft Excel
- **Skills demonstrated:** Data cleaning, handling missing values, 
  removing duplicates, data validation, conditional formatting

### Cleaning Steps
- Identified and handled missing values using pivot tables and MODE formula
- Removed duplicate records

- Applied data validation to prevent invalid entries
- Documented cleaning process with data dictionary reference

## Cafe Sales Dashboard
- **Tool:** Microsoft Excel
- **Dataset:** Dirty Cafe Sales Dataset (Kaggle)
- **Skills demonstrated:** Data cleaning, pivot tables, dashboard design,
  slicers, charts

### Key Insights
- Digital Wallet is the most preferred payment method
- Juice is the top selling item by revenue
- Short distance customers purchase more frequently
- Middle aged customers drive the most purchases
